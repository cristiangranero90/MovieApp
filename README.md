## Movie App
### A simple movie visualizer

1. Hilt, dependency injection
2. Coroutines
3. Retrofit 2
4. Room Database
5. ViewModel
6. Jetpack Compose
7. Material Design
8. Gson serialization
9. Light & Dark mode available
10. MVVM project
